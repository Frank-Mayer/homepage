# My Homepage

CI: ![Vercel](https://vercelbadge.vercel.app/api/Frank-Mayer/homepage)

## Run dev server

1. Install the dependencies
   ```bash
   npm install
   ```
1. Start the dev server
   ```bash
   npm run dev
   ```
1. Open the browser at http://localhost:3000

## Build for production

1. Install the dependencies
   ```bash
   npm install
   ```
1. Build the project
   ```bash
   npm run build
   ```
