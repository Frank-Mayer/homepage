docdown

Automated documentation tool.

The library is used to make writing the text as easy as possible. Custom plug-ins for this editor allow linking sources and creation of a table of contents and source lists.

The app is an installable PWA. Offline usage as well as automatic updates work without any problems. Recent documents are automatically saved in the browser's IndexedDB and can be accessed via a menu.

The finished document can then be exported as a PDF document using the library.

Photon

Simple and efficient Web frontend framework.

Subpage router, custom templates, (data binding is currently in progress).

No special web server is needed, because everything is executed on the client-side. The framework consists of a JavaScript runtime (also useable by itself) and a CLI tool that pre-renders subpages and templates, to ensure search engines can see the content and the page loads even faster in the clients browser. Server-side logic is not provided.

If the CLI tool is used the page will work without JavaScript. A bundler (like Parcel) must be executed before the CLI tool. Also a robots.txt and sitemap.xml will be created automatically if not already present.

Compatible with any HTML5 capable browser (with limited functionality even without JavaScript).
