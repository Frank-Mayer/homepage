docdown (WIP)

Automatisierted Dokumentationstool.

The Editor.js library is used to make writing the text as easy as possible. Custom plug-ins for this editor allow linking sources and creating a table of contents and source listing.

The app is an installable PWA. Offline usage as well as automatic updates work without any problems. Recent documents are automatically saved in the browser's IndexedDB and can be accessed via a menu.

The finished document can then be exported as a PDF document using the PDFMake library.

Photon

Simple and efficient web frontend framework

Subpage Router, Custom Components, Data-Binding.

No special web server is needed, because everything is done client-side. However, the router requires that the /index.html is always loaded, if the requested file was not found (404 can be served using the framework).

Earliest compatible browser versions: Chrome 58 (Jan 2017), Edge 14 (Aug 2016), Firefox 54 (Mar 2017), Safari 10 (Jul 2016), Opera 55 (Aug 2018).
