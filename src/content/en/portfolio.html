docdown (WIP)

Automated documentation tool.

The library is used to make writing the text as easy as possible. Custom plug-ins for this editor allow linking sources and creating a table of contents and source lists.

The app is an installable PWA. Offline usage as well as automatic updates work without any problems. Recent documents are automatically saved in the browser's IndexedDB and can be accessed via a menu.

The finished document can then be exported as a PDF document using the library.

Photon

Simple and efficient web frontend framework

Subpage Router, Custom Templates, Data-Binding.

No special web server is needed, because everything is done client-side. However, the router requires that the /index.html is always loaded, if the requested file was not found (404 can be served using the framework).

Earliest compatible browser versions: Chrome 58 (Jan 2017), Edge 79 (Jan 2020), Firefox 63 (Oct 2018), Safari 10 (Sep 2016), Opera 55 (Aug 2018).
