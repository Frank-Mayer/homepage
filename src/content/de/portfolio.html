docdown (WIP)

Automatisierted Dokumentationstool.

Die Bibliothek wird verwendet, um das Schreiben des Textes so einfach wie möglich zu gestalten. Eigene Plug-ins für diesen Editor ermöglichen das Verlinken von Quellen sowie das Erstellen eines Inhaltsverzeichnisses und einer Quellenauflistung

Die App ist eine installierbare PWA. Offlinebetrieb sowie automatische Updates funktionieren problemlos. Die letzten Dokumente werden automatisch in der IndexedDB des Browsers gespeichert und können über ein Menü aufgerufen werden.

Das fertige Dokument wird dann mit der Bibliothek als PDF-Dokument exportiert.

Photon

Simples und effizientes Web Frontend Framework

Subpage-Router, Templates, Data-Binding.

Es wird kein spezieller Webserver benötigt, da alles clientseitig ausgeführt wird. Der Router erfordert jedoch, dass immer die /index.html geladen wird, wenn die angefragte Datei nicht gefunden wurde (404-Seite kann über das Framework eingestellt werden).

Kompatibel mit jedem Browser (mit eingeschränkter Funktionalität auch ohne JavaScript)!
