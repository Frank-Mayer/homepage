docdown (WIP)

Automatisierted Dokumentationstool.

Die Bibliothek Editor.js wird verwendet, um das Schreiben des Textes so einfach wie möglich zu gestalten. Eigene Plug-ins für diesen Editor ermöglichen das Verlinken von Quellen sowie das Erstellen eines Inhaltsverzeichnisses und einer Quellenauflistung.

Die App ist eine installierbare PWA. Offlinebetrieb sowie automatische Updates funktionieren problemlos. Die letzten Dokumente werden automatisch in der IndexedDB des Browsers gespeichert und können über ein Menü aufgerufen werden.

Das fertige Dokument wird dann mit der PDFMake Bibliothek als PDF-Dokument exportiert.

Photon

Simples und effizientes Web Frontend Framework

Subpage-Router, Custom Components, Data-Binding.

Es wird kein spezieller Webserver benötigt, da alles clientseitig ausgeführt wird. Der Router erfordert jedoch, dass immer die /index.html geladen wird, wenn die angefragte Datei nicht gefunden wurde (404 kann über das Framework eingestellt).

Frühesten kompatiblen Browser-Versionen: Chrome 58 (Jan 2017), Edge 79 (Jan 2020), Firefox 63 (Okt 2018), Safari 10 (Sep 2016), Opera 55 (Aug 2018).
